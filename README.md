## How to run 実行方法

1. download the repo リポジトリをダウンロードする

2. Install the dependencies:

```
npm install
```

3. Run the site:

```
npm start
```

4. Open the site in your browser:

```
http://localhost:3000
```

(You can test with the CSV files provided in the Example CSVs Folder)
